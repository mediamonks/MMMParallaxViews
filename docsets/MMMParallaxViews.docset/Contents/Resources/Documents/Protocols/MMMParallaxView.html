MMMParallaxView

A view that can be placed over a Scroll/TableView that follows a certain point or cell. The height can be resized on scroll, so you get a parallax effect; e.g. resize a header to be a sort-of navigation bar. Or resize a footer when the bottom is reached to show more detailed information.

  • parallaxView Default implementation

    The view whose position and size will be managed.

    Default Implementation

    For compatibility with the existing code where parallax views have to conform to MMMParallaxView.
